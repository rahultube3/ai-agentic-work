# Web Scraper

Extract data from web pages.

## Capabilities
- Static HTML parsing (CSS and XPath selectors)
- Headless browser rendering for JS-heavy sites
- Pagination and infinite scroll handling
- Configurable rate limiting and retries
- Output to JSON, CSV, or Parquet
