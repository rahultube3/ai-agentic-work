# MCP Builder

Bootstrap MCP server projects.

## Capabilities
- Generate Python or TypeScript MCP server scaffolds
- Define tool handlers with JSON schema validation
- Define resource providers
- Local stdio and HTTP transport support
- Example tests and CI configuration
